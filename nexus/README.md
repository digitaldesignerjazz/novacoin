# Nexus Orchestration Integration

This directory is for all code, schemas, and documentation related to deep integration between the live Nexus orchestrator and the NovaCoin blockchain.

Planned contents:
- Task schemas for on-chain bounties
- Event listeners / webhooks from chain to Nexus
- Dynamic parameter update mechanisms
- Self-improvement feedback loops
- Bridge monitoring and reconciliation tasks

Nexus is the central intelligence layer that makes NovaCoin a living, self-evolving system.
