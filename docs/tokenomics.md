# NovaCoin Tokenomics

## Overview
NovaCoin is the base-layer settlement and coordination token of the NovaNet ecosystem. It works in tandem with the QCoin rune layer (particularly NovaRune) to create a multi-dimensional economy.

## Key Parameters
- **Ticker**: NOVACOIN
- **Decimals**: 6 (or as defined in genesis)
- **Initial Genesis Supply**: 1,000,000,000,000,000 base units
- **Inflation**: Initially 0% (dynamic via Nexus self-improvement)
- **Utility**: Gas, staking, settlement, prototype funding, agent rewards, bridge liquidity

## Economic Layers
1. **Base Settlement (NovaCoin)**: Core value transfer and staking
2. **Infrastructure Rune (NovaRune)**: Mesh relay rewards, node operation incentives
3. **Creative & Ritual Layer**: Meaning-driven minting and consumption

## Dynamic Emission
Nexus continuously adjusts emission, staking rewards, and task bounties based on:
- Network health (mesh performance, validator uptime)
- Agent swarm productivity
- Prototype milestone completion
- Creative output quality

This creates a true self-improving economy rather than fixed monetary policy.
