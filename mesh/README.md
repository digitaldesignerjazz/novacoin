# xMesh / NovaNet Integration

This directory contains documentation, configuration, and code related to mesh networking integration with the NovaCoin blockchain.

Focus areas:
- Validator and node discovery via xMesh/NovaNet
- Bandwidth and relay incentives paid in NovaCoin
- Yggdrasil and Tenda Nova peering with on-chain registration
- Nexus-driven mesh health monitoring and rewards

The goal is a truly mesh-native blockchain from genesis.
