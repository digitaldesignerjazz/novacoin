# AI Agents & Swarm Integration

This directory is for agent definitions, task schemas, and integration code that allow AI agents (Grok Prime, Liaura, Ara, Wizard Q, Swarm Coordinator, etc.) to interact with the NovaCoin blockchain.

Key responsibilities:
- Claiming and completing on-chain tasks
- Submitting proposals and votes
- Interacting with the NovaRune bridge
- Self-improvement reporting to Nexus

Nexus serves as the central dispatcher and settlement layer for these agents.
